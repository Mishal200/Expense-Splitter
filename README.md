Expense Splitter 💸

A simple and responsive Expense Splitter Web App built using HTML, CSS, and JavaScript. This project allows users to add expenses, view the total amount, edit entries, and delete expenses dynamically.

📌 Features

➕ Add new expenses

📝 Edit existing expenses

❌ Delete expenses

💰 Automatically calculates total expenses

📱 Simple and responsive UI

⚡ Built with pure JavaScript

📂 Project Structure

Expense-Splitter/

│

├── index.html # Main HTML structure

├── script.css # Styling for the application

└── script.js # JavaScript functionality
